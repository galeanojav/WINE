Weighted-Interaction Nestedness Estimator (WINE)


WINE is a GUI Matlab application developed to perform the calculation of the new weighted nestedness estimator. This program permits to: open an Excel file, select the Excel range of data that is to be analysed, and calculate the results, showing two graphics and three indexes. The indexes shown in the graphics interface are: a) weighted nestedness of the frequency matrix, b) weighted nestedness of the random matrix, and c) the new weighted-interaction nestedness estimator. The GUI provides a histogram of the frequency of interactions and a figure showing a color plot of the packed matrix, in such a way that every link is plotted by a color corresponding to the weighted distance.


In contrast with previous nestedness estimators, WINE is not based on presence-absence adjacency matrix. This program takes into account the frequency in which link occurs to calculate the index of nestedness.
